<h1> Read: 05 - Design web pages with CSS </h1> 

1. ***What is the purpose of CSS?***
It separates content (HTML) from presentation (design)


2. ***What are the three ways to insert CSS into your project?***
- 1️⃣ Inline CSS-Written directly inside an HTML element.

- 2️⃣ Internal CSS-Written inside a <style> tag in the <head> section of the HTML file.

- 3️⃣ External CSS ✅ (Best practice)-Written in a separate .css file and linked to the HTML.


3. ***Write an example of a CSS rule that would give all <p> elements red text.***
- Example of a CSS rule that makes all <p> elements red
p {
  color: red;
}
p → selector

color → property

red → value
